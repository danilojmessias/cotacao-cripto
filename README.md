# cotacao-cripto
Aplicação para ver cotação de criptomoedas em tempo real

## Autor

[danilojmessias](https://www.github.com/danilojmessias)

## Rodar Localmente

Clone o Projeto

```bash
 https://github.com/danilojmessias/cotacao-cripto.git
```


Instale as dependencias

```bash
  npm install
```

Inicie o Projeto

```bash
  npm run start
```
